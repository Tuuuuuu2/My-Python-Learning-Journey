### 心理学系统
这是一个包含两个独立Python程序的集合，主要用于初级的心理学相关的娱乐与数据管理。
## 项目概述
本项目包含两个独立的Python程序：
心理学灵力检测系统：评估研究人员的学术生产力
研究团队管理系统：管理心理学实验人员分配
### 快速开始
## 环境要求
Python 3.6+
无需额外依赖包
## 运行方式
直接运行Python文件：
python "2023341026刘佳.py"
## 程序详细介绍
1. 心理学灵力检测系统
# 功能描述
通过分析研究人员在2023-2025年期间的SCI论文发表情况，评估其学术生产力水平。
# 核心功能
数据输入：收集三年的SCI论文发表数
质量筛选：确保每年都有论文产出
统计分析：计算年均发表量，找出最佳和最差年份
等级评定：根据生产力水平给出评级（小牛/牛/牛中牛）
2. 研究团队管理系统
# 功能描述
一个完整的CRUD系统，用于管理不同心理学实验的研究人员分配。
# 核心功能
添加研究人员：为指定实验添加新的研究人员
查看所有人员：显示所有实验及其对应人员
查找特定人员：按实验名称查询相关人员
退出系统：安全退出程序
